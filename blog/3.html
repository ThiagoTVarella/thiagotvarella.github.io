Running local LLMs on CPU

→ Read the original

Open source models have a relatively narrow capability gap compared to proprietary models. The greatest barrier for running these models locally is the price to acquire GPUs with memory big enough to be able to run these models. This link compiles some strategies and interesting explanations/engineering to make a model runnable on CPUs instead. I didn't find a benchmark and it looks like it's still not fast enough for a reasonable agentic workflow, but important to keep track of nevertheless.
